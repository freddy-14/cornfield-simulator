# cornfield-simulator

SMS based cornfield simulator.

## Configure
Copy `config.tml` to `corn.yml` and modify as you see fit.

## Build
```
$ mvn package
```

## Run
```
$ java -jar target/cornfield-simulator-0.1.jar server config.yml
```
